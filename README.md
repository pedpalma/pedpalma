## 📌 Projetos em Destaque

### ⚙️ [ISP Manager (em desenvolvimento)](https://github.com/pedpalma/isp_manager.git)

Sistema para gerenciamento de ONUs e OLTs via API. Automatiza provisionamento e monitoramento de equipamentos de rede, reduzindo trabalho manual da operação.  
**Stack:** Python/FastAPI backend com Celery, SQLAlchemy, structlog, PostgreSQL, e Redis; Next.js frontend.

### 🌐 [G-Lab Website](https://github.com/pedpalma/G-Lab_website.git)

Landing page institucional da G-Lab Telecom, com foco em apresentação da marca e captação de leads.  
**Stack:** HTML, CSS, JavaScript e Vercel.

### 📱 [Chicos Burger App](https://github.com/pedpalma/chicos-burger-app.git)

Aplicativo para Hamburgueria Chico's Burger, direcionado a apresentar o cardápio e a história do restaurante, disponibilizando também uma página de pedidos e uma página de cadastro.
**Stack:** React Native, TypeScript e JavaScript.
